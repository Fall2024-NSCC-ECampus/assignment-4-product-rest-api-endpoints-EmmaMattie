Courier Service Management System
Welcome to the Courier Service Management System project! This system is designed to streamline the processes involved in managing courier services, including product orders, payments, and delivery tracking.

Project Overview
This project is a simple, efficient solution to manage daily courier activities. It provides a platform to:

Browse and purchase products.
Handle product returns.
Process payments.
Display company and hub details.
Track deliveries and non-delivery information.
View and leave product reviews.

Features
Product Listings: Users can view a list of available products.
Purchase Products: Allows users to buy products directly through the platform.
Return Products: Handles product returns smoothly.
Payment Window: Simple and secure payment processing.
Company Details: Shows basic information about the courier company.
Hub Rates: Displays rates for different hub locations.
Delivery Details: Provides tracking information for delivered items.
Non-delivery Details: Shows information on items that couldn't be delivered.
Product Reviews: Users can read and add reviews for products.

How to Use
Clone the Repository: Download or clone this repository to your local machine.
Build and Run: Open the project in Intellij, build it, and run the program.
Explore the Features: Try the various features listed above to see how the system works.

Future Scope
This system is designed to serve as a foundational project for courier management systems. With further development, it can expand to include additional features, like advanced tracking, personalized user accounts, and support for multiple payment methods.
